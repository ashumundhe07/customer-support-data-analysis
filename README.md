# 📊 Customer Support Data Analysis Dashboard

## 🚀 Project Overview

This project analyzes customer support ticket data to identify trends, improve resolution efficiency, and track SLA performance.

## 🛠 Tools Used

- Power BI
- Excel
- SQL (Basic)

## 📂 Dataset

The dataset includes customer support tickets with details such as:
- Issue type
- Priority level
- Status
- Resolution time
- Agent information

## 📊 Key Insights

- ✅ Identified most frequent issue types affecting users
- ✅ Analyzed SLA breach percentage and resolution efficiency
- ✅ Tracked ticket trends over time
- ✅ Highlighted high-priority issue patterns
- ✅ Identified top-performing agents and resolution bottlenecks

## 📈 Dashboard Features

- 🎯 Interactive Power BI dashboard with drill-down capabilities
- 🔍 Filters (Issue Type, Priority, Agent, Date Range)
- 📊 KPI metrics (Total Tickets, SLA Breach %, Avg Resolution Time, First Response Time)
- 📉 Time-series analysis of ticket trends
- 🏆 Agent performance tracking
